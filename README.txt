# Quizle: Immersive Learning Explorer

Quizle is a state-of-the-art educational platform designed to transform structured data into a rich, interactive learning experience. It leverages AI-generated JSON modules to provide a seamless transition from reading to mastery.

## 🎯 Project Goal
The goal of Quizle is to provide a "one-stop shop" for learning any complex topic. By importing a specific JSON schema (generate-able via the included `system_prompt.txt`), users can instantly access:
- **Interactive Materials**: Beautifully rendered markdown content for deep reading.
- **Active Recall Flashcards**: 3D-flippable cards to drill core concepts.
- **Adaptive Quizzes**: Challenging assessments with real-time feedback, hints, and detailed explanations.
